# Logo Assets Directory

This directory contains logo variations for different use cases.

## Required Logo Files

Create the following logo variations:

### 1. **Logo Variations**
- `logo-light.png` - White text version for dark backgrounds
- `logo-dark.png` - Dark text version for light backgrounds  
- `logo-icon.png` - Just the fork & spoon icon (no text)

### 2. **Logo Sizes**
- `logo-200.png` - 200px wide (for headers, small displays)
- `logo-400.png` - 400px wide (for medium displays)
- `logo-800.png` - 800px wide (for large displays, print)

## Usage Guidelines

### When to Use Each Version:

**logo-light.png:**
- Dark backgrounds
- Hero sections with dark overlays
- Footer (if dark background)

**logo-dark.png:**
- Light backgrounds
- White/cream colored sections
- Print materials

**logo-icon.png:**
- Favicons
- Small spaces
- Social media profile pictures
- Watermarks

### Size Guidelines:

- **200px**: Headers, mobile navigation
- **400px**: Hero sections, feature sections
- **800px**: Large displays, print materials, high-res displays

## Creating Logo Variations

1. Open `public/logo.png` in your design tool
2. Create variations as needed
3. Export with appropriate sizing
4. Optimize file sizes (< 100KB each)
5. Save to this directory

## File Naming Convention

- Use lowercase with hyphens
- Include size in filename for sized versions
- Be descriptive (light, dark, icon)

---

*Logo files should be created from the source logo.png file using design software like Photoshop, Illustrator, or Figma.*
